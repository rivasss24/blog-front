## Requisitos
Debe inicializar el [servidor](https://github.com/rivasss24/blog-back) para continuar

## Clonar proyecto

```
https://github.com/rivasss24/blog-front.git
```

## Instalar dependencias

```
cd ./blog-front
npm install
```

## Inicializar proyecto

```
npm run dev
```
